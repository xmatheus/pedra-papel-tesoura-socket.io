`download repo`
> git clone [https://github.com/xmatheus/pedra-papel-tesoura-socket.io](https://github.com/xmatheus/pedra-papel-tesoura-socket.io)
> 
`change dir`
> cd pedra-papel-tesoura-socket.io

`install and exec`
> yarn

> yarn dev
